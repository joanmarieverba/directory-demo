# Directory-demo
The purpose of this respository is to store code for a directory app.

Written in ReactJS using Aphrodite, Horizon, Elemental UI among others

Features:
Responsive design -
Login with ID -
Alphabetical directory -
Search function -
Print/save function -
Confirmation page -
Ability for Admin to delete records
